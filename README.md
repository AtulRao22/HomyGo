# 🏡 HomyGo – Property Listing Platform

HomyGo is an Airbnb-inspired web application that allows users to explore and list properties with ease.

## 🚀 Features
- 🏠 Property listing system
- 🔍 Browse available properties
- 🧾 User-friendly interface
- 📦 Backend integration with database

## 🛠️ Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: MongoDB

## 📌 Purpose
This project was built to enhance my full-stack development skills and understand real-world application flow.
